# TODO LARAVEL API

## Deployment
1. clone this repo
2. cd to repo folder
3. open terminal and run ```cp .env.example .env```
4. modify the env to match your local db, url etc
5. Run ```composer install```
5. Run ```php artisan key:generate```
5. Run ```php migrate:fresh```
